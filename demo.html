minimal carousel - demo

  • First impressive message!
  • Second message with all the magic.
  • Also, you can configure minimal carousel as you want.

Usage

HTML

  <div class="carousel">
    <ul>
      <li>
        First impressive message!
      </li>
      <li>
        Second message with all the magic
      </li>
      <li>
        Also, you can configure minimal carousel as you want.
      </li>
    </ul>
  </div>

CSS

    .carousel ul li {
      display: none;
    }

Javascript

    new Carousel({
      carousel: '.carousel',
      delay: 2.5,
      autoplay: true
    })
